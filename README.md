# Formula 1 Data Analysis

## Project Overview
Analyzed Formula 1 race and qualifying data using SQL and Python.

## Tools Used
- Python
- Pandas
- SQL
- Matplotlib

## Questions Answered

1. Does qualifying position affect race finish?
2. Which drivers outperform their qualifying position?
3. Which teams are most consistent?

## Methods

- Data Cleaning
- SQL Joins
- Group By Aggregations
- Linear Regression

## Key Findings

- Strong positive relationship between qualifying and race results.
- Certain drivers consistently gain positions during races.
